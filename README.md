## Setup & Run

### Prerequisites:
Client & server
- NodeJS version: 16 (not supported: 17)
- Web browser: Chrome, Mozilla Firefox

<br />  
<hr>
<br />  

### Installation

#### React & Nodejs
To install the dependencies for both `/frontend` & `/backend` folders followed by database initialization<br>

### Run


#### Server

`cd backend/`  

`docker-compose up -d`  

`npm run start:dev`  

#### Client

`cd frontend/`  

`npm run dev`
